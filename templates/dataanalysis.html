Fig.1: Distribution of data under various Flairs.
  • Distribution of data under various Flairs.


    Distribution of data under various flairs is shown in this donut.


      Data is collected from PRAW: The Python Reddit API Wrapper for various flairs for instance: “Food”, “Coronavirus”, “Business/Finance”, “AskIndia”, “Sports”, “Science/Technology”, “Scheduled”, “Politics”, “Policy/Economy”, “Photography” and “Non-Political”. It is ensured that proportion of data taken remains even in all 11 flairs.

      Data is preprocessed before feeding it into classifier using two processing steps:

    • Punctuation marks like “,”, “!”, “?” etc. are removed using regular expressions in python.

    • Stop words like “the”, “is” etc. are removed using Spacy Library in python.
